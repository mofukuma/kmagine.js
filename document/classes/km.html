load

(
  • url
)

Defined in kmagine.js:103

限りなくユルく使える素材ロード

Parameters:

  • url String/object/array

    読み込むデータのURLまたは設定オブジェクト、またはそれらの配列

Example:

km.load("images/foo.png");

km.load("images/foo.png", "images/bar.jpg");

km.load(["images/foo.png", "images/bar.jpg"]);

km.load(["images/foo.png", "images/bar.jpg"], function(){...onload callback...} );

km.load("images/foo.png", "sounds/se.mp3", "sounds/bgm_130.mp3"); // filename_XXX.mp3など、末尾に数字を追加することで、自動ループとbpmが指定できる

km.load({src:"images/foo.png"}, {src:"sounds/se.mp3"}, {src:"sounds/bgm_130.mp3", bpm:130}); // 連想配列で詳細指定することも可能。これは１つ上と同じ処理をおこなう
