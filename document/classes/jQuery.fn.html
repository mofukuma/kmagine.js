to

(
  • params
  • [duration]
  • [easing]
  • [callback]
)
chainable

Defined in kmagine.js:1604

くまじんのメインとなるトゥーインアニメ関数。 jQueryプラグインとして動作する。使い勝手はjQuery animateと同じ。 内部でCSS3 transition-propertyを使用しているためGPUに乗り高FPSでヌルヌル動く。 他、loop～endloopによる繰り返しアニメや、 よくあるアニメを呼び出せるプリセット機能を搭載。

Parameters:

  • params Object

    CSSに設定する

  • [duration] Int optional

    アニメの長さ（ms）

  • [easing] String optional

    アニメ補完の種類

  • [callback] Function optional

    アニメ終了後に実行されるコールバック関数を指定

Example:

$("#foo").to({x: 0, y: 10}, 1000);

$("#foo").to({x: 0, y: 10}, 1000, "swing", callback_fn );

$("#foo").to( "shake", 400 ); //プリセット。400msの間、揺らす

$("#foo").to( "poyo" ); //ぽよんとする。生物感を出すためのアニメ

$("#foo").to( "scaleOut" ); //だんだん透明になりつつ大きくなっていく

$("#foo").to( "scaleIn", 450 ); //上記の逆を450msで
